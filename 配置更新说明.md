## 🎯 支持的AI模型

### 默认模型
- `anthropic/claude-3.5-sonnet`（当前默认）

### 可选模型
修改 `api/_utils/aiService.js` 中的 `DEFAULT_MODEL`：

```javascript
// 使用GPT-4 Turbo
const DEFAULT_MODEL = 'openai/gpt-4-turbo';

// 使用Claude 3 Opus
const DEFAULT_MODEL = 'anthropic/claude-3-opus';

// 使用Gemini Pro
const DEFAULT_MODEL = 'google/gemini-pro';
```

查看所有可用模型：https://openrouter.ai/models
