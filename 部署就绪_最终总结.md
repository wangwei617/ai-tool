## 🚀 部署步骤（简化）

### 1. GitHub上传（2分钟）

```bash
# 进入项目目录
cd "/Users/modao/Documents/2026 Q1 project/AI使用方法/场景设计方案/完整应用"

# 初始化Git
git init
git add .
git commit -m "Initial commit: AI增强产设研工作平台 - Vercel版本"

# 在GitHub创建新仓库后
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git branch -M main
git push -u origin main
```

### 2. Vercel部署（2分钟）

1. 访问 [vercel.com](https://vercel.com)，GitHub登录
2. 点击 "Add New..." → "Project"
3. 选择GitHub仓库
4. 配置环境变量：`ANTHROPIC_API_KEY = your_api_key`
5. 点击 "Deploy"

### 3. 完成！

访问Vercel提供的URL即可使用。
