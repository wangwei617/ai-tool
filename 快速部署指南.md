## 🎯 三步部署

### 步骤1：上传到GitHub（2分钟）

```bash
# 1. 进入项目目录
cd "/Users/modao/Documents/2026 Q1 project/AI使用方法/场景设计方案/完整应用"

# 2. 初始化Git仓库
git init

# 3. 添加所有文件
git add .

# 4. 提交
git commit -m "Initial commit: AI增强产设研工作平台"

# 5. 在GitHub创建新仓库（网页操作）
# 访问 https://github.com/new
# 仓库名：ai-work-platform（或自定义）
# 不要勾选 "Initialize with README"

# 6. 添加远程仓库（替换YOUR_USERNAME和YOUR_REPO）
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git

# 7. 推送到GitHub
git branch -M main
git push -u origin main
```

### 步骤2：部署到Vercel（2分钟）

1. **访问Vercel**
   - 打开 [vercel.com](https://vercel.com)
   - 使用GitHub账号登录

2. **导入项目**
   - 点击 "Add New..." → "Project"
   - 选择你的GitHub仓库
   - 点击 "Import"

3. **配置项目**
   - Framework Preset: **Other**
   - Root Directory: `./`（默认）
   - Build Command: **留空**
   - Output Directory: **留空**（Vercel自动识别）

4. **配置环境变量**
   - 展开 "Environment Variables"
   - 添加变量：
     - Name: `ANTHROPIC_API_KEY`
     - Value: 你的Claude API密钥
   - 确保勾选所有环境（Production、Preview、Development）
   - 点击 "Add"

5. **部署**
   - 点击 "Deploy"
   - 等待1-2分钟
   - 部署完成！

### 步骤3：验证部署（1分钟）

1. **访问应用**
   - 部署完成后会显示URL
   - 点击URL访问应用

2. **测试功能**
   - 测试健康检查：`https://your-domain.vercel.app/api/health`
   - 测试各个功能模块
   - 确认功能正常

## 🔧 常见问题

### Q1: 部署失败 - 找不到模块
**A:** 
```bash
# 确保package.json包含所有依赖
cd 项目目录
npm install
git add package.json package-lock.json
git commit -m "Update dependencies"
git push
```

### Q2: API调用失败 - 401错误
**A:** 
- 检查Vercel Dashboard中的环境变量
- 确保 `ANTHROPIC_API_KEY` 已配置
- 重新部署项目

### Q3: 前端无法加载
**A:**
- 检查 `public/` 目录是否存在
- 检查文件路径是否正确
- 查看浏览器控制台错误

### Q4: 文件上传失败
**A:**
- 检查文件大小（建议<4MB）
- 检查浏览器控制台错误
- 确认API路由正确
